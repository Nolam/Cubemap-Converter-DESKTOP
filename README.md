**Cubemap Converter (Desktop)** is an open-source, free-to-use, portable tool developed by David Parrella to convert cubemap (or "skybox") format textures into equirectangular HDRI images in multiple formats and sizes.
Created specifically with SpaceEngine in mind, this software can reassign axes to convert from non-traditional coordinate systems.


**How to use:** <br>
* Navigate to [Releases](https://github.com/Nolam/Cubemap-Converter-DESKTOP/releases) and download the latest portable version<br>
* Extract it to the desired location (such as your Desktop), and run it on your local machine<br>
* Select the Single DDS cubemap file, or switch to the individual faces mode to upload individual face files<br>
* Choose "SpaceEngine" or the appropriate coordinate system (for the software that generated the cubemap)<br>
* Select the desired resolution and file format, then export<br>
<br>
Note: You may have to ignore safety prompts the first time you run the program.
